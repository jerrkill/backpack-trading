# auto_trade_backpack_exchange


Auto trade on Backpack exchange with Javascript:


- index.js auto trade
- grid trading



# usage

```
cp .env.example .env

```

config .env

```

API_SECRET = ""
SYMBOL_PAIR = "JUP_USDC"
GRID_CENTER_PRICE=
NUMBER_OF_GRIDS=
CAPITAL_PER_GRID=
```


```
node ./autoGrid.js
```
